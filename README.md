# Customer Behaviour and Shopping Trend Dataset Analysis

## Overview
This project analyzes customer shopping behavior to identify purchasing patterns, customer segments, and product trends.  
It demonstrates an end-to-end data analytics workflow using **Python (Pandas), PostgreSQL, and Power BI**.

---

## Dataset
**Name:** Customer Behaviour and Shopping Trends Dataset  

**Description:**  
The dataset contains customer demographic information, purchase details, product categories, discounts, subscription status, shipping types, and review ratings.

**Key Columns Include:**
- Customer ID, Gender, Age Group  
- Item Purchased, Category  
- Purchase Amount, Discount Applied  
- Shipping Type, Subscription Status  
- Review Rating, Previous Purchases
